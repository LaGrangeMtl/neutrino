Neutrino Demo

As you can see, if you inspect the source code, Neutrino is a very simple markup based slideshow.

I used sections and articles, but you can use divs as long as they have the classes neutrino for the container and slide for each slide, you're good to go!

The rest of your efforts will be concentrated in the JavaScript.

No worries. It's easy. I promise.

You're gonna need 3 things.

  • var neutrinoSettings = {};
  • var neutrino = Neutrino();
  • neutrino.init(neutrinoSettings);

This is where the fun begins

You see that variable we created right there? neutrinoSettings ?

Well in that variable, you can set every settings of Neutrino. And, it's amazingly simple!

As of version 0.1.0, here are the settings you can edit for Neutrino :

  • transitionTime
  • timer

You set them like that :

var neutrinoSettings = {
transitionTime: 1,
timer: 2000
}

Detailed info on the settings

transitionTime

This setting requires a number. It is the number of seconds that a transition takes to complete. You can go from 0 to whatever you want. Except it's gonna be long and you're gonna get mad for waiting too long. True story bro.

timer

This setting requires an integer. It is the number of MILLISECONDS at which rate the transitions will take place. For example, if I set 2000, it will take 2 seconds between each transition of the slideshow. Be careful of always thinking in milliseconds for this one.
